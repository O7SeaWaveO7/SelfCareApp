# Claude.md

## Project Overview

This project is a mobile self-care habit tracking app focused on helping users build consistency through habit tracking, visual streaks, and gamification.

The app should feel:

- clean
- motivating
- modern
- lightweight
- visually rewarding

The core goal is to make habit tracking addictive in a healthy way through streaks, progress visualization, and rewards.

---

# Core Features

## 1. Daily Habit Tracking

Users can:

- create custom habits
- edit habits
- delete habits
- mark habits as completed daily

Examples:

- Drink Water
- Exercise
- Read
- Journal
- Sleep Early

Requirements:

- habits reset every day
- quick one-tap completion
- smooth completion animations
- minimalist interaction flow

---

## 2. Calendar Visual Streaks

The app should strongly emphasize visual progress.

Features:

- calendar view for each habit
- completed days highlighted
- missed days visible
- current streak counter
- longest streak tracker

The streak system should:

- reward consistency
- motivate users to avoid breaking streaks
- make progress visually satisfying

Design inspiration:

- GitHub contribution graph
- Duolingo streak system
- modern productivity apps

---

## 3. Gamification

The app should include lightweight gamification elements.

Features:

- XP / points for completed habits
- streak bonuses
- level progression
- achievements / badges

Example achievements:

- First Habit Completed
- 7-Day Streak
- 30-Day Streak
- 100 Habits Completed

The reward system should feel:

- motivating
- satisfying
- non-overwhelming

Avoid excessive complexity.

---

# Design Direction

## UI Style

- minimal
- modern
- soft colors
- calming aesthetic
- rounded cards
- clean typography
- smooth animations

## User Experience

The app should be:

- extremely easy to use
- fast for daily check-ins
- visually rewarding
- optimized for mobile

A user should be able to:

1. open app
2. complete habits
3. view streaks
4. close app

within seconds.

Preferred Stack:

- React Native

- TypeScript

- React Navigation
- Reanimated

# Important Constraints

DO NOT include:

- social media features
- chat systems
- AI features
- journaling
- mood tracking
- complicated onboarding

Keep the project focused ONLY on:

- habits
- streaks
- gamification
- visual consistency tracking
